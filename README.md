### Survive In The Bee Hive

Survive the bees attack wave there are only 2 types of bees yellow and red you need to dodge both bees if player gets hit game is over, here player needs to catch only the red bees through mouse and click if you try to catch yellow game will be over, there will be total 3 waves of bees attack each wave increases the number of both bees.


### Controls

Keyboard & Mouse:
 - SPACE to start.
 - A W S D to move.
 - Mouse to click
 - Tab to pause
